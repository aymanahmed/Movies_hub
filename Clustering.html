Clustering

Cluster analysis or clustering is the task of grouping a set of objects in such a way that objects in the same group (called a cluster) are more similar (in some sense or another) to each other than to those in other groups (clusters), It is a main task of exploratory data mining, and a common technique for statistical data analysis, used in many fields, including machine learning , pattern recognition, image analysis, information retrieval, bioinformatics, data compression, and computer graphics,There are a number of different methods that can be used to carry out a cluster analysis; these methods can be classified as follows: Hierarchical method and Non-hierarchical methods (often known as k-means clustering methods).
